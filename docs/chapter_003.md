# 编程高手  
    
## 第3章 绩效考核  
    
难得一个周末不上班，这在创业型的小公司是非常珍贵的。孩子去上培训班了，老婆也出去采购了。郭靖懒洋洋地醒来，看到温暖的阳光马上快到床沿了。  
    
有数据统计，所有的成功人士，都是每天只睡四到六小时的。反正，数据显示，睡得越少越伟大。比如，撒切尔夫人，一天只睡三小时。苹果公司的乔布斯，现在的库克，微软的比尔盖茨，甚至还有推特的创始人，脸书创始人等等，他们基本都是4到6小时。  
    
如果拿郭竞跟这些人比，根本不在一个档次。人家个个亿万富翁，但郭竞仍只是一个每日要为生计写代码的人。不过，他却是真的阅读过《睡眠革命》这本书。并且，在他读完以后，微微一笑，心想，“所有这些，我都做了20多年了。”  
    
确实如此，郭竞周一到周五，每天只睡5小时，只有周六和周日才九小时。这样，凭空他就比别人每周多出10多个小时。  
    
郭竞躺在床上，在回忆昨晚在网上翻阅的ZeroMQ和protocol buffer。新技术对他来说总有吸引力，不只是使用，更有项目的形气意声值得他学习。  
    
在他看来，编程就像练功一样。一个好的程序也有其形气意声所谓的形，则是代码的用词与语句。比如，曾经有个开源中，一个类的静态初始化函数名为of，让他拍案叫绝。因为确实简洁，并且意思明了，基于英语法，我就是字符串初始化的对像，或json初始化的对像，或数组初始化的对像。of足够且是绝佳函数名。对于其意字，含义却非常深刻了，关键还看写代码的人的状态，比如Carbon，月日时分秒随意加减，不会有-1月，13月，-1分，60分。这样的代码才算是形到，意到。  
    
郭竞在思考ZeroMQ的形意问题，这是他昨晚九点到今天凌晨两点学习的内容，可就在这时，电话响了。原来，后端小伙伴因为一个绩效需求搞不定，去公司加班了，可仍是搞不定，电话向他求救了。这种事情一要在过去遥控是无法解决的，但现在，凭借强大的互联网，还是有办法的。所以，郭竞电话里跟对方说，“你先把代码提交，我一会看一下。”  
    
说完，即去洗潄，然后吃早餐。  
    
郭竞家的早餐都是自家做的。上周他也休息。所以自己动手做了包子。看到餐桌上自己做的包子，便一下子来了胃口。因为，他做的包子与中国著名的狗不理，富春等品牌包子的外形几呼没什么区别。而且，他们家崇尚吃素，并且拥有独有的香菇青菜馅的配方，绝对的美味。  
    
其实，这也缘于多年多以前，他在朋友的饭店做义工。他当时的目的不只是帮忙，而是要学一些本领。所以，他现在包饺子，是全家人中速度最快的一个。不仅会做包子，还会做一手好菜。  
    
等郭竞打开笔记本，SVN更新完代码，他觉得比较崩溃。实际上，需求是这样的。销售团队，每个人除了个人销售业绩，同时要负责带团队。分为三级，经理，主管，销售员。经理之上就是销售部总监。经理，主管，销售员均要考核所带的人数。除了上下级关系，还有平级帮带。在做数据表设计时，通过带着大家讨论，最终设计是，使用位存储模式保存绩效标准和实际绩效。但真正做时，他的这位小伙伴只做了几个公用函数，即拆成数组，合成整数等。其它一切全散落在业务中了。  
    
郭竞叹了一口气。然后，在lib目录新建了一个子目录ByteNumber，并新建了一个类，名称也是ByteNumber。然后，就像行云流水一样开始了他的编码：  
    
构造函数，接受数组参数;  
    
静态实例化函数of,接受长整型整数;  
    
然后是加法add 按级加法，addByGrade;  
    
同样还有减法 subtract，subtractByGrade 乘法 multiply。  
    
应当也有除法，divide 因为，需要算完成百分比。他这样想着，于是代码就成写出来了。  
    
然后，比较大小，CompareTo,基于此，由会有条件判断，于是，他写了greaterThan，greaterThanOrEqualTo,lessThan,  
    
最后则是转换函数，toArray，toInt，toString。  
    
所有一切几乎是一气呵成。  
    
并且，所有运算符号，完全返回对象本身。方便链式操作。方便写算式。花了不到半小时测试，然后在考核业务代码中做了两处修改，就像  
    
if($achievement->greateThanOrEqualTo($criteria))  
    
我想，各位看官，如果你会编程，你也会惊呼他这美妙的设计吧。也许你也会说，不就是个计算层分离吗。是的。棋高一着，缚手缚脚！大师就是不一样。  
    
差不多完工，郭竞电话中给小伙伴讲解了一下。然后，开始制作他的周六家庭午餐了。  
    
而这时，微信中许丽敏发来一个动画表情：谢谢你！郭竞回了一个微笑。
    
但没想到的是，这时电话却响了。来电显示，许丽敏。郭竞按下接听，即听到：
    
“郭竞，有急事找你，麻烦你走一趟，过来一起吃饭，具体面谈。地址微信发给你了，你到了直接跟服务员说，是我订的座位就可以了。好了，我在开车，先挂了”。
    
再看微信的定位，陆家嘴的避风塘。郭竞心想，这都什么逻辑，我要是不去呢?你怎么知道我闲着?尽管如此，郭竞还是给老婆打了电话，说明了情况，然后驱车前往。
    
当服务员把他带到预订的座位，郭竞发现，许丽敏身边还有一位帅哥。许丽敏开始相互引见。
    
“来来来，我先给你们介绍一下，这位就是我跟你说过的郭竞。文会琴棋书画，精通中国古文化，比如，周易，术数，还懂中医养生。武也能散打，摔跤，计算机与互联网方面，绝对的编程高手。”
    
“不要听她吹了，我懂得不多，所有都是入门级的。”郭竞赶紧给自己圆场。

“自己人，就没必要谦虚了。这位是韩博，记得我以前跟你说过。并且，托你的福，让我看了《不要一个人吃饭》，所以，今天，我就这样把你拉来了。韩博刚刚还说，我这样打电话，你可能不会过来。最终，你还是来了。韩博这次是来海市出差，本打算昨天回去的，我向他介绍了你，他就把机票改签到今晚了。”
    
“多谢，多谢。首先，还是要声明一下，我没有许丽敏吹得那么神。一个小公司的CTO，根本不算什么。”
    
“是这样，郭总，任何人都是靠实力打天下的。只有你强了，别人才会觉得你可依赖。我想，许总不会看错人。所以，我专门留下来，也是为了能够和你一起吃个饭，交流一下。”
    
“第一，是想讨教你在技术学习方面的诀窍。第二，则是讨教一些管理方面的问题。应当没有问题吧?”郭竞听韩博说到这里，深感佩服。牛人就是牛人，效率为上，直接切入正题。

“肯定没问题，你在大公司，应当是我们向你学习才对。再说了，对于管理，许总可比我强多了。”郭竞回复道。

“来来来，边吃边聊。郭竞你就先介绍一下你学技术的诀窍吧”，许丽敏招呼道道。

“没问题，我先班门弄斧，给个最精简的版本，十个字：学上乘功夫，为自己做事。”
    
“给你先点个赞。能否解释一下呢?”韩博问道。
    
“武打片都看过吧，皅自己当成武林中人，每学一点技术，就是多学会一套武术套路。有目标地学习上乘功夫，这样的心态之下肯定学得快。其次，员工是要有绩效考核的，如果自己的定位不是员工，那么，做开发就是练武，一切都对自己成长有利，所以，时刻记住，为自己做事。”
    
“不错不错，听你这话，感觉到，天下所有编程高手几乎都是这样产生的。”韩博称赞道。
    
“也不完全，不同的经历会造就不同的人。我和许丽敏最关心的问题是，如何把一个公司做大、做强。韩总能否指导一下?”
   
“你们这个话题好大，这可不像我刚刚问你的问题，几句话就能说完。”
    
“现在还有时间，你反正是晚上的飞机，不妨根据现有的时间简单指导一下。”许丽敏说。
    
“简单来说，要想把公司做强做大，分为以下几个方面，品牌，产品，团队，市场。对于品牌，很多人可能认为无所谓。不过我们看到，能成功的都有一个好品牌。比如，腾讯飞信，这其中有几个品牌?”
    
“两个。”许丽敏说。
    
“三个，是腾讯，讯飞，飞信”。
    
“我们可以猜测，后面两个品牌利用了关联效应。或许，讯飞一开始想叫飞讯也有可能。再举个例子，拉扎斯公司肯定是几乎无人知道的，但是，它的品牌——饿了么现如今却是无人不知的。外卖这个行业，现如今百度也挤了进来。我们可以分析一下这三个品牌。饿了么表现了对用户的关爱，而美团则沿袭了团购的便宜与实惠，都有好的品牌寓意。至于百度，技术牛人从不用百度，百度外卖这四个字，首先就失掉了一批最有效且最频繁使用的用户。就像你我，根本不以你是大公司，就认为可信赖。其次，这四个字与前面二者，单从文字寓意也是不能比的。百度，一百度，肯定是太烫了，吃不成的。而有百度的暗示，冬天，很多用户收到外卖时肯定抱怨，饭凉了。所以，从这三个品牌来看，饿了么肯定是最好的品牌。或许，我们都能预测，最后胜出的只会是饿了么与美团。”
    
“说到产品，无非就是简单的服务，关键看你这个服务是否有关爱，是否贴心。顾客是上帝，这句话我们不知说了多少年，但有多少产品以此为中心，实际并不多。就像现在，我们为什么放弃米聊，都改用微信。好用是最根本的前提。这里有个关键，一是像米聊那样过于绅士，不碰你的通讯录，于是，米聊中没你的朋友，无法用。最终导致米聊因为用不了而成了根本不好用。二是像脉脉那样强制导入用户的所有通讯录。最终，用户不外乎是同意与不同意。而我们能够想到，不愿导入的人，不外乎几种理由。第一，自我保护较强。不想泄露给你。第二，属于善良的技术人，不想出卖朋友给你骚扰。微信的做法，则是把决定权交给你，你是上帝。当然，优质服务还有很多细节，比如，产品易用性的问题，百度外卖本次升级前，普通人会不会用，你我心中都有数。假如一个产品首先是把心思用在营销与赚钱，而忽视了服务，这样的产品是做不长的。”
  
“产品不多说了，因为，产品本身与创始人以及产品经理均有关联，也就是说核心问题则是人，也就是团队问题。说起团队，我个人认为，创始人的格局很重要，比如，京东所以是京东，那是因为他能抓住电商机会做物流。当然，一个公司，不外乎是找一帮人来做好一件事。把这个过程分解下来，最关键最核心是，如何忽悠小伙伴把事情做出来，并且做得尽可能快，尽可能好。像郭竞你这样的人，自然不需要绩效考核。但我们面对的是九零后，实际上我们时刻都觉得，要让他们明白过来，难度是很大的。其实，这并不是中中特有的问题。全世界都有，特别是发达国家。我就听过国外一些合作公司的抱怨。对此，最有名的创新，那就是Google公司的OKR绩效考核了。你们听说过吗？”
    
许丽敏点头，郭竞说，“是的，关键是如何落地”。
    
“OKR落地有障碍的公司，都是比较难以做大的。原因不外乎几点，第一，产品不够阳光，第二，创始人格局不大，核心团队想多挣钱。第三，产品方向不明。第四，核心团队管理观念落后 。另一方面，OKR考核方法目前只有英文资料，这说明国内还没有足够重视。我觉得，未来在中国肯定会出现基于OKR绩效管理咨询的公司，甚至会有相关的软件。”
    
“是个不错的商机。”郭竞说道。
    
“说起Google，有一本书，叫《重新定义公司》，你们都看过吗？”
    
“正在看”郭竞回答。
    
“有机会我也看看。”许丽敏说。
    
“一个公司的企业文化如何建立，如何让员工认同，公司应当提供什么样的成长氛围，此书中都有很好的解答。”
    
“而对于团队绩效考核，股权激励也是必不可少的。这也与创始人格局有关。股权期权员工如果挣到了，企业挣得会更多，但很多创始人看不到这一点的。”
    
“最后是关于市场。这里所说的市场，是指市场、销售与营销的全部。有些团队融资，常说部里有什么关系，省里有什么关系，其实，这只是市场因素中极小的一部分。品牌，产品，团队比起单一的这个因素要大得多。营销再强，产品不好，也是白费。比如，易信，它就没有解决好友关系如何建立的问题。市场也不只是市场人的事情，比如产品有没有传播特性，也决定了市场的难度。比如，美图秀秀，就是一个传播力很强的产品。产品也要适合时代，适应当前的新科技。日落西山的产品，再好的营销也不会有用户。比如，前两年的‘狗日的腾讯’这个事件，因为这件事，《计算机世界》报确实火了，但是，当人们注意力不断被手机蚕食的年代，报纸要能生存，并不是一个高明的营销事件，或者干一些有情怀的事情就能做到的。尽管《计算机世界》的这件事可能会名垂青史，但却不能增加订户。空有社会责任感也是没有用的。”
    
“刚刚说到的市场是关于用户这一方面的。另一关键的方面就是对手，很多企业都是一不留神就给对手干掉了。比尔盖茨说过，微软离破产永远只有十八个月。互联网相关新兴行业的企业更是这样，因此，企业融资是必不可少的，因为，没有资本方的参与，就能难快速成为行业的独角兽。也就很难在一个行业坐稳。现今的百度，腾讯，阿里都经历了这样的过程。”
    
“总的来说，要想把公司做大，没有哪个环节容易。产品不好做，大家有目共睹。看似相当简单的社交产品，为什么微信能成，并不只是刚才所说的解决了社交关系建立的问题。有人说团队最难，确实，无论找人还是招人，别人来跟你一起做，要的是成功，创始人如何让大家看到希望，并有信心跟你一起做，这又涉及到你的商业计划，财务模型，当然，如果你是雷军这样的大神除外。人来了，怎样把事情做好，我认为，这更加重要。很多A轮死，B轮死，甚至是C轮死的企业，多数死在管理上。而管理，就是要大家心往一处想，力往一处使，力争投入最低，产出最大。往简单里说，就是一个绩效考核问题。往复杂里说，则是一个系统工程。这也就是，BAT公司出来的，很多创业成功了，而中兴，华为出来的则是十创九不成，没有一个做大的，原因很简单，生产型企业的管理与互联网企业完全不同，大公司与小公司管理也完全不同。我见过中兴公司出来，自己公司直接套用中兴的流程，使用中兴的制度，采用中兴的管理方法。后来，当然，不是倒了，就是维持一个小作坊。所以，如果要把公司做大，本质上不是以自己有多牛自居。而是要能以最省钱与最省时的办法解决问题，这才是最重要的。企业永远有两缺，一是缺钱，二是缺人。大多数企业家都把各种没有做好归因为这两缺。这种心态如何做大?”
    
“你说得没错，创始人的格局与心态实际上也很重要，是不是一起做一个好的事业，造福用户，还是只是为了挣钱。我见过好几个团队，开始相当好，挣钱后反而团队散了。没有足够的阳光的心态，是完全做不好企业的。”郭竞似乎有感而发。
    
这时，许丽敏在微信中建了一个群。并建议郭竞与韩博加一下微信。二人响应，互加完成。
    
韩博问道：“郭总有没有考虑去大公司，或者，这样说吧，我现在缺一个像您这样的手下。我这边，大公司，大平台，保证你有晋升空间。如何?”
    
郭竞心想，这才是你留下来的本意呀。于是问道，“做什么呢?”
    
“第一，我现在的关键项目组的项目经理很不得力，所以，准备换掉他。第二，目前在申请一个类似于电子杂志的项目。如果项目下来了，将交给你全权负责。”
    
“我目前CTO也快不干了，将由新人接替。”
    
“这不正好吗？”
    
“是这样的，公司按我的要求成立员工培训学院，我将负责这一块了。这是我想干的。并且我已答应了。”郭竞推辞道。
    
“这样吧，我回去先落实一下你的待遇，后续把明确的待遇给你，然后，你再给我答复。”韩博认为，不就是钱吗，钱能解决的问题，都不是问题。
    
许丽敏却突然感到若有所失，她没想到，现在只是觉得郭竞可能会走，就是这样，竟然让她心中如此伤感。这是怎么了。郭竞倒底走不走，居然是她目前心中最迫切要明确的问题。这场饭局结束后，许丽敏做了一个决定。但郭竞却一无所知。
